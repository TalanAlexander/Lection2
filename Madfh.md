# Инструкции по работе с Марком

## Выделение текста

Чтобы выделить текст курсивом, необходимо заключить его (*). Например *Курсив*

Чтобы выделить текст полужирным, необходимо заключить его в двойные звёздочки (**). Например **Полужирный**
